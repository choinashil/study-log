- Part8. 조건문
  - switch문
```
switch ( A ) {
    case 1:
        실행될 문장;
        break;
    case 2 :
        실행될 문장;
        break;
    default :
        실행될 문장;
        break;
}
```
- 
  - case 뒤에 1,2..는 A와 비교할 값 
  - case ~ 뒤에는 ;(세미콜론) 아니고 : (콜론)!!
  - case - if / else if 역할
  - default - else 역할 
  - break가 없다면 순차적으로 모든 코드 실행
